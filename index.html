Welcome to Sui-Gen

Sui-Gen is a precision lunisolar calendar generator that fetches high-precision astronomical data from the NASA JPL Horizons API (DE440/DE441 ephemerides) to calculate exact New Moon timings and Solar Terms, following Yuk Tung Liu's Chinese Calendar algorithm. Each year is enriched with the zodiac animal, Wu Xing element, and Ganzhi designation, with full leap-month detection. An offline fallback is available if the API is unreachable.

Astronomical Precision

Sui-Gen integrates directly with the NASA JPL Horizons API, fetching high-precision DE440/DE441 ephemerides data (and interpolating) to calculate the exact timing of New Moons and Solar Terms.

Hard Data Limits

Because JPL Horizons dataset boundaries are strictly adhered to, highly accurate calendar generations are limited to the years between 619 CE and 17191 CE. Generating outside this range will return blank fields for astronomically-dependent parameters.

Cycle Stability

Lunisolar cycle fields (Zodiac, Elements, Ganzhi) are calculated via continuous pure arithmetic and are 100% reliable infinitely for any year.

Reference: Calculation of the Chinese Calendar (Yuk Tung Liu)
